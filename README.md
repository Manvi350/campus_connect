# 📘 Campus Connect – College Management System

## 📌 Overview
Campus Connect is a web-based college management system designed to streamline academic activities and communication between students, teachers, and administrators. It provides a centralized platform to manage attendance, assignments, study materials, marks, and notices efficiently.

---

## 🚀 Features

### 🎓 Student Panel
- View profile and personal details  
- Check attendance records  
- Access class schedules  
- View assignments and study materials  
- Check marks and results  
- View notices  

### 👨‍🏫 Teacher Panel
- Manage student attendance  
- Upload assignments  
- Upload study materials  
- Enter sessional marks  
- Post notices  

### 🛠️ Admin Panel
- Manage students and teachers  
- Manage classes and timetable  
- Generate reports  
- Control system settings  

---

## 🧩 Tech Stack
- **Frontend:** React.js  
- **Backend:** FastAPI (Python)  
- **Database:** MongoDB  

---

## 🔄 System Flow
The system follows a centralized architecture:

1. Users (Student, Teacher, Admin) interact with the frontend  
2. Requests are sent to backend APIs  
3. Data is processed and stored in MongoDB  
4. Responses are returned and displayed on the UI
